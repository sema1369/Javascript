Javascript - Les différents types de données

1/ Les trois types primitif principaux en JS

  1. String ou chaine de caractères
  2. Number ou nombre
  3. Boolean ou booléen

Pour connaitre les types de données on utilise la fonction typeof()

String

ce contenu sera remplacé

Les chaînes de caractères (chaînes, ou strings, en anglais) sont la façon d'enregistrer du texte dans des variables JavaScript. On peut enregistrer dans une variable de type string n'importe quelle chaîne de caractères, allant d'une seule lettre à un très grand nombre de lettres (plus de 134 millions, même dans des navigateurs anciens).
Les variables de type string sont encadrées par des guillemets simples '...' ou doubles "..." :
Exemple : let lastName = 'Alexander'; / let firstName = "Will";

En javascript, on va pouvoir utiliser indifféremment des guillemets doubles ou des quotes pour entourer une chaîne de caractères et ces deux méthodes vont être strictement équivalentes à la différence d’autres langages comme le PHP par exemple.
Faites attention cependant à un point :
si votre chaîne contient un caractère qui est le même que le délimiteur de chaîne choisi, il faudra neutraliser ce caractère en l’échappant au moyen d’un antislash ou changer de délimiteur.

Les chaînes peuvent aussi être concaténées (ajoutées à la fin l'une de l'autre) par l'opérateur + :
Exemple :
let wholeName = firstName + " " + lastName; //valeur: "Will Alexander".

Il est possible depuis quelques années d’utiliser une nouvelle écriture qui simplifie la concaténation des variables et des chaînes de caractère : la string interpolation.
Pour créer une string interpolation on écrit du texte encadrée par le signe ` et si on veut injecter une variable dans ce code on utilise l’expression ${maVariable}.
Exemple :
const myName = `Alexander`;
const salutation = `Bienvenue sur mon site ${myName}!`;
console.log(salutation); //retournera “Bienvenue sur mon site Alexander!”

Number

Les variables en javascript vont également pouvoir stocker des nombres. Des nombres entiers (1, 2, 3, etc.) positifs ou négatifs ou à virgule, décimaux (1,4 ; 67,34 ; etc.).
En programmation, les nombres entiers sont aussi appelés entiers ou integers ; les nombres avec des chiffres après la virgule sont aussi appelés nombres en virgule flottante ou floating-point.
Exemple :
let integerCalculation = 1 + 2; // donne 3
let weirdCalculation = 0.1 + 0.2; // on attend 0.3, réponse réelle 0.30000000000000004

Chaque fois que c'est possible, utilisez plutôt des calculs entiers (pour le calcul de prix, vous pouvez par exemple calculer en centimes, et non pas en euros ou en dollars).

Boolean

Ce type de données sera utile pour verifier si un test est vrai ou faux.

Un booléen, en algèbre est une valeur binaire (soit 0, soit 1).
En informatique, les valeurs logiques (booleans) sont le plus simple des types primitifs : le type booléen est un type qui ne contient que deux valeurs : true (vrai) et fasle (faux).
Elles s'utilisent dans toutes sortes de cas : pour indiquer si un utilisateur est connecté ou non, si une case est cochée ou non, ou si un ensemble de conditions particulières est réuni.
Exemple : let userIsSignedIn = true; / let userIsAdmin = false;
