12 - Les opérateurs logiques

Pour effectuer plusieurs comparaisons.

Ce type d'opérateurs va nous permettre d'effectuer plusieurs comparaisons dans nos tests ou d'inverser la valeur logique du résultat d'un test.

Nom Symbol Description
AND (ET) && renvoie true si toutes les comparaisons sont vraies ou false sinon.
OR (OU) || renvoie true si au moins l'une des comparaisons est vrai, ou false sinon.
NO (NON) ! renvoie false si une comparaisons est évalué à true
