2 - Les variables

Qu'est-ce qu'une variable ?

Une variable est un conteneur (boite) servant à stocker des informations (du texte, un nombre, un nom d'utilisateur, le nombre de billets restants pour un vol ou la disponibilité d'un certain produit en stock) de manière temporaire. Toutes ces données sont enregistrées dans des variables.
Une variable peut varier, elle stocke différentes valeurs au fil du temps.

Règles de déclaration d'une variable

On "déclare" une variable avec le mot-clef : let

  • Le nom d'une variable doit obligatoirement commencer par une lettre ou un underscore _
  • Le nom d'une variable ne doit contenir que des lettres, des chiffres et des underscores _
  • Le nom d'une variable doit avoir une signification précise.
  • Il n'y a pas d'espace dans le nom d'une variable.

Initialiser une variable

Lorsque l'on assigne une valeur à une variable vide on dit que l'on stocke une valeur ou qu'on initialise une variable.

Modifier la valeur d'une variable

La façon la plus simple de modifier la valeur d'une variable est simplement de la réaffecter.

La portée d'une variable

La portée d'une variable désigne l'endroit où elle va pouvoir être utilisée dans un script.
