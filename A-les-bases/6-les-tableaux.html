6 - Les tableaux (arrays)

Utiliser un tableau pour enregistrer une liste ordonnée d'éléments

a) Pour créer un tableau vide et l'enregistrer dans une variable, utilisez une paire de crochets : let invites = [];

b) Vous pouvez aussi créer un tableau rempli en plaçant les éléments voulus à l'intérieur de ces crochets :
let invites = ["Kemi seba", "Nathalie yamb", "Hash carter"];

c) Vous pouvez ensuite accéder aux éléments de ce tableau par leur indice : N'oubliez pas de démarrer à l'indice 0 !
let premierinvites = invites[0]; // résultat : "Kemi seba"
let secondinvites = invites[2]; // résultat : "Hash carter"
let invitesNonDefini = invites[12] // résultat : undefined

Travailler sur les tableaux

d) Le comptage d'éléments dans un tableau avec la propriété length

La propriété length d'un tableau indique le nombre d'éléments qu'il contient :
let invites = ["Nelson Mandela", "Malcom X", "Frantz Fanon"];
let nombreInvites = invites.length; // résultat : 3
Utilisez la notation dot pour accéder à la propriété length de votre tableau. Ce n'est peut-être pas très utile dans cet exemple (parce que nous savons combien nous avons mis d'invités dans le tableau !), mais dans beaucoup de cas, vous ne saurez pas à l'avance combien il y a d'éléments dans un tableau.

e) L'ajout d'éléments dans un tableau avec la méthode push

Pour ajouter un élément à la fin d'un tableau, utilisez la méthode push :
invites.push("Marcus Garvey"); // résultat : ajoute "Marcus Garvey" à la fin du tableau invites.

f) L'ajout d'éléments dans un tableau avec la méthode unshift

Pour ajouter un élément au début d'un tableau, utilisez la méthode unshift :
invites.unshift("Jerry Rowlings"); // résultat : ajoute "Jerry Rowlings" au début du tableau invites.

g) Suppression du dernier élément d'un tableau avec la méthode pop

Pour supprimer le dernier élément d'un tableau, utilisez la méthode pop , sans passer aucun argument :
invites.pop(); // résultat : supprime le dernier élément du tableau

Conclusion

On sait désormais : Créer des tableaux, les remplir, et les manipuler avec certains outils de base.
