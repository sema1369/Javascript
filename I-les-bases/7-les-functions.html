7 - Les functions

Les functions en JS sont incontournables

Qu'est-ce qu'une fonction ?

Une fonction est un bloc de code nommé et réutilisable, et dont le but est d'effectuer une tâche précise ; les fonctions possèdent de nombreux atouts.

  • Il existe 2 types de fonctions en JS : les fonctions natives ou prédéfinies qu'on n'aura qu'à appeler. Et les fonctions personnalisées qu'on va créer.
  • On crée une fonction personnalisée grâce au mot-clef function :
  • Pour exécuter le code d'une fonction il faut l'appeler. Pour cela, il suffit d'écrire son nom suivi de parenthèses en passant éventuellement des arguments dans les parenthèses.
  • Si une fonction a besoin qu'on lui passe des valeurs pour fonctionner, alors on définira des paramètres lors de sa définition.
  • Lors de son appel, on lui passera des arguments qui prendront la place des paramètres.

Déclaration d'une fonction

On "déclare" une fonction avec le mot-clef : function

Exemple :
// On déclare la fonction
function afficheDeuxValeurs(valeur1, valeur2) {
console.log('Première valeur:' + valeur1);
console.log('Deuxième valeur:' + valeur2);
}
// On exécute la fonction
afficheDeuxValeurs(12, 'Bonjour');
// résultat : On obtient dans la console.log
// Première valeur : 12
// Deuxième valeur : Bonjour

Beaucoup de fonctions ont besoin de variables pour effectuer leur travail. Quand vous créez ou déclarez une fonction, vous indiquez la liste des variables dont elle a besoin pour effectuer son travail : vous définissez les paramètres de la fonction.
Ensuite, à l'appel de la fonction, vous lui attribuez des valeurs pour ses paramètres. Les valeurs sont les arguments d'appel. Enfin, votre fonction peut vous donner un résultat : une valeur de retour.
