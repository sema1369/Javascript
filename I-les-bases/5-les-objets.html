5 - Les objets

Qu'est-ce qu'un objet?

Un objet est un ensemble cohérent de propriétés et de méthodes.

Les objets en JavaScript sont écrits en JSON (JavaScript Object Notation).
Ce sont des séries de paires clés-valeurs séparées par des virgules, entre des accolades. Les objets peuvent être enregistrés dans une variable :
let personne = {
prenom: 'Sema',
nom: 'Stg',
age: 27
};
console.log(personne);

Chaque clé est une chaîne (prenom, nom, age...), et les valeurs associées peuvent avoir tout type de données (chaîne de caractère, nombre, etc.).
Construire des objets présente un avantage essentiel : cela permet de regrouper les attributs d'une chose unique à un même emplacement, que ce soit un livre, un profil d'utilisateur ou la configuration d'une application, par exemple.

Comment accéder aux données d'un objet avec dot notation ?

Maintenant que vous savez comment créer un objet en JavaScript, voyons comment accéder aux données dans un objet avec la notation pointée (dot notation), expliquée ci-dessous.
Une fois qu'un objet est enregistré dans une variable, vous pouvez accéder à ses données comme dans l'exemple ci-dessous :
let monLivre = {
titre: "L'Histoire de Tao",
auteur: "Will Alexander",
nombreDePages: 250,
estDisponible: true
};

let titreDuLivre = monLivre.titre; // résultat : "L'Histoire de Tao"
let pagesDulivre = monLivre.nombreDePages // résultat : 250

Pour cela, utilisez le nom de la variable qui contient l'objet, ensuite placez un point ' . ', puis le nom de la clé dont vous souhaitez récupérer la valeur.

La notation bracket (bracket notation)

Pour accéder à un sous élément vous pouvez utiliser les brackets avec la valeur du sous élément.
Exemple :
let monLivre = {
titre: "L'Histoire de Tao",
auteur: "Will Alexander",
nombreDePages: 250,
estDisponible: true
};

let titreDuLivre = monLivre["titre"]; // résultat : "L'Histoire de Tao"
pagesDulivre = monLivre["nombreDePages"]; // résultat : 250

L'intérêt ici est de pouvoir mettre entre bracket une variable qui contient en string le nom de la propriété que l’on souhaite atteindre.
Exemple :
let monLivre = {
titre: "L'Histoire de Tao",
auteur: "Will Alexander",
nombreDePages: 250,
estDisponible: true
};

let accesPropriete = "titre";
titreDuLivre = monLivre[accesPropriete]; // résultat : "L'Histoire de Tao"
