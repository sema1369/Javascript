Javascript - conclusion partie 2

Dans cette partie du cours, nous avons parlé de la logique de programme. Nous avons vu :

  1. Certains environnements où JavaScript peut être utilisé, et la façon dont son exécution peut être différente d'un environnement à l'autre ;
  2. Les instructions conditionnelles telles que if/else et switch ;
  3. Les boucles pour des sections répétitives de code, telles que for et while ;
  4. Les différents types d'erreurs, et comment les traiter.
