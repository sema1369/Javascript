Javascript - Les variables et leurs valeurs

1) Qu'est-ce qu'une variable ?

Une variable est un conteneur (boite) servant à stocker des informations de manière temporaire, du texte (une chaîne de caractère), un nombre, un nom d'utilisateur, le nombre de billets restants pour un vol, la disponibilité ou non d'un certain produit en stock, toutes ces données sont enregistrées dans des variables.. La variable varie ! Elle stocke différentes valeurs au fil du temps.

Une variable et une valeur sont deux choses différentes.

a. Règles de déclaration

On "déclare" une variable avec le mot-clef : let

  • Le nom d'une variable doit obligatoirement commencer par une lettre ou un underscore _
  • Le nom d'une variable ne doit contenir que des lettres, des chiffres et des underscores _
  • Il n'y a pas d'espace dans le nom d'une variable.

b. Initialiser une variable

Lorsque l'on assigne une valeur à une variable vide on dit que l'on stocke une valeur ou qu'on initialise une variable.

c. Modifier la valeur d'une variable

La façon la plus simple de modifier la valeur d'une variable est simplement de la réaffecter.

d. La portée des variables

La portée d'une variable désigne l'endroit où elle va pouvoir être utilisée dans un script

2) Qu'est-ce qu'un opérateur arithmétiques ?

Un opérateur est un symbole qui va effectuer certaines opérations sur les variables et leurs valeurs. Par exemple l'opérateur va permettre de multiplier, additionner ou soustraire les valeurs de deux variables.

Tableau des opérateurs arithmétiques
Opérateurs Nom de l'opération
+ Addition
- Soustraction
* Multiplication
/ Division
% Modulo
** Exponentielle
Modulo
Le reste entier d'une division euclidienne. On divise 5 par 3 le résultat est 1 et il reste 2 : le reste est le modulo
Exponentielle
L’exponentielle correspond à l’élévation à la puissance d’un nombre par un autre nombre :
Exemple : 23

a. Priorité des calculs et utilisation des parenthèses.

Si on écrit let h = 1 - 2 - 3; le résultat est -4 car les opérations se font de gauche à droite.

A l'exception de l'élévation à la puissance qui sera calculée en partant de la fin.
Exemple : let k = 2 ** 3 ** 2; qui correspond à 2 3 = 8 puis 8 3 = 512

Nous allons finalement, comme en mathématiques, pouvoir forcer l'ordre de priorité en utilisant des parenthèse pour indiquer qu'une opération doit être effectuée avant les autres.
Exemple : let l = (1 - 2) * 3;

b. Les opérateurs d'affectation

Les opérateurs d'affectation vont nous permettre d'affecter une certaine valeur à une variable ;

Egal "=" est l'opérateur le plus utilisé.

Opérateurs d'affectation
Opérateurs Définition
+= Additionne puis affecte le résultat
-= Soustrait puis affecte le résultat
*= Multiplie puis affecte le résultat
/= Divise puis affecte le résultat
%= Calcul le modulo puis affecte le résultat

3) Les constantes

Une constante est similaire à une variable au sens que c'est un containeur qui contient une valeur, mais on ne pourra jamais modifier la valeur d'une constante !

Dans de nombreux cas, certaines données ne seront pas modifiées. C'est le cas par exemple du nom d'une entreprise, de la date de naissance d'un utilisateur, ou du nombre d'heures dans une journée. Pour s'assurer de ne pas réaffecter par inadvertance de nouvelles valeurs à ces données, on utilise des constantes.

a. Règles de déclaration

Pour déclarer une constante en JS on utilise le mot clef const. On déclare une constante en JS comme une variable, par contre il faut obligatoirement l'initialiser lors de sa déclaration.

Conclusion

Dans ce chapitre, on a apprit les bases des variables en JavaScript.

On sait désormais :

  • Déclarer les variables par le mot clé "let" et un nom en "camelCase", et les initialiser avec l'opérateur = ;
  • Modifier le contenu d'une variable en la réaffectant, ou avec des opérateurs ;
  • Utiliser des constantes pour éviter le remplacement d'éléments de données essentiels.
